== Description ==
Plugin desarrollado para integrar la API de senders.com.ar en tu Woocommerce.

Sobre Senders,
Envia y recibe lo que quieras en el día.
Hacé envios en el día desde tu computadora o celular. Decinos por donde vamos a buscar tu pedido y a donde lo entregamos y nosotros nos hacemos cargo del resto.

== Installation ==

1. Install the WooCommerce Senders - Plugin.
2. Activate the plugin.
3. Go to the WooCommerce Settings Page. 
4. Access Shipping Tab.
5. Select "Senders".
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use.
